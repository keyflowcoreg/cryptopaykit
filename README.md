# CryptoPayKit

x402 crypto payment integration toolkit for Next.js.

[![Live Demo](https://img.shields.io/badge/Live_Demo-cryptopaykit.vercel.app-blue?style=for-the-badge)](https://cryptopaykit.vercel.app)

[Live Demo](https://cryptopaykit.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/cryptopaykit/issues)

## Features

- 15 ready-to-use code templates for x402 payment integration
- 4 step-by-step integration guides
- Live interactive playground to test payment flows
- Templates for Next.js, Express, and more
- USDC on Base -- no credit cards, no Stripe

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion · x402 Payments

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4003](http://localhost:4003) to view it locally.
